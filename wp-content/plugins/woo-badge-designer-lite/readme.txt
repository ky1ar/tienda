<strong>Badge Designer Lite For WooCommerce</strong>  is your ideal WordPress plugin to transform the default WooCommerce sale badges into more appealing badge designs. This advance plugin helps you add text, image and icon labels to your products in a jiffy. And the labels that can be added via this plugin come in super handy when there is any sale, discount, special features, unique product information.

== Installation ==
1. Unzip woo-badge-designer-lite.zip
2. Upload all the files to the /wp-content/plugins/woo-badge-designer-lite.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. For customizing the plugin's settings, click on Badge Designer Lite For WooCommerce option menu in Wordpress left admin menu.

== Frequently Asked Questions ==
= How many badges can be created for Woocommerce? =
You can create unlimited number of badges.

= Is this plugin compatible with all themes?  =
This plugin is compatible with most themes that follow the development standards of WordPress.And also it should have woocommerce generated products.

= Can we change the color and fonts of the badges? =
Yes, you can put your own color and background color for badges.t.

= How do I use Badge Designer Lite For WooCommerce? =
First of all you have to create the badges and then you have to assign badges to each product from Badge Designer Lite For WooCommerce Settings tab in product page.
